# Quantum, Physics & Materials

> [!abstract] Scope
> Quantum computing frameworks, open quantum systems, astronomy, fluid dynamics, and materials science.

[Back to Skill Index](../index.md)

**Related maps:** [Data Science, Stats & Scientific Computing](data-science-compute.md) | [Machine Learning & AI](ml-ai.md)

## Skills (7)

- [astropy](../astropy.md) — Core Python library for astronomy and astrophysics workflows that need Astropy APIs, including units/quantities, coordinates, FITS I/O, tables, time systems, WCS, and cosmology
- [cirq](../cirq.md) — Google quantum computing framework. Use when targeting Google Quantum AI hardware, designing noise-aware circuits, or running quantum characterization experiments
- [fluidsim](../fluidsim.md) — Framework for computational fluid dynamics simulations using Python
- [pennylane](../pennylane.md) — Hardware-agnostic quantum ML framework with automatic differentiation
- [pymatgen](../pymatgen.md) — Materials science toolkit. Crystal structures (CIF, POSCAR), phase diagrams, band structure, DOS, Materials Project integration, format conversion, for computational materials science
- [qiskit](../qiskit.md) — IBM quantum computing framework. Use when targeting IBM Quantum hardware, working with Qiskit Runtime for production workloads, or needing IBM optimization tools
- [qutip](../qutip.md) — Quantum physics simulation library for open quantum systems
