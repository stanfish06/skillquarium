# molecular-docking

> [!info] What it does
> Classical, physics-based protein-ligand docking with AutoDock Vina (and smina, plus GNINA for CNN-rescoring). Use for receptor/ligand prep (Meeko, OpenBabel, RDKit), defining the search box (center + size), running docking, interpreting affinity scores, pose analysis, virtual screening, and rescoring DiffDock poses with GNINA. Trigger terms: "docking", "AutoDock Vina", "smina", "GNINA", "protein-ligand", "virtual screening", "binding pose", "PDBQT", "dock this ligand", "rescore poses".
